## 💳 Sistema de Gestão de Contas a Receber (Controle de Fiado)

## 📌 Sobre o projeto

Este projeto foi desenvolvido com o objetivo de aplicar, na prática, conceitos de programação em Java e lógica de backend.

A aplicação simula um sistema simples de controle de dívidas (fiado), comum em pequenos comércios, permitindo registrar, acompanhar e gerenciar valores em aberto de clientes.

Mais do que apenas fazer o sistema funcionar, o foco foi estruturar bem o código, aplicar orientação a objetos e desenvolver uma base sólida para evolução futura.

---

## 🚀 Funcionalidades

* Cadastro de dívidas com cliente (nome e telefone)
* Listagem de todas as dívidas
* Exibição de status (PENDENTE / PAGO)
* Marcar dívida como paga
* Cálculo do total de valores em aberto

---

## 🧠 Conceitos aplicados

* Programação Orientada a Objetos (POO)
* Encapsulamento (uso de atributos privados + getters/setters)
* Relacionamento entre classes (Cliente → Dívida)
* Estrutura de dados (`ArrayList`)
* Controle de fluxo (`if`, `switch`, `for`)
* Manipulação de entrada de dados com `Scanner`
